Internal Certificate Authority

This is the root certificate for the .x network. Install and trust it so your device accepts the certificates served by internal services (e.g. https://andromeda.x) without warnings.

Download root_ca.crt

Installing

  • Linux (Debian/Ubuntu): 
    curl -fsSL http://ca.x/root_ca.crt -o /usr/local/share/ca-certificates/x-root-ca.crt
chmod 644 /usr/local/share/ca-certificates/x-root-ca.crt
update-ca-certificates
  • Linux (Arch): 
    curl -fsSL http://ca.x/root_ca.crt -o /etc/ca-certificates/trust-source/anchors/x-root-ca.crt
chmod 644 /etc/ca-certificates/trust-source/anchors/x-root-ca.crt
update-ca-trust
  • macOS: open in Keychain Access → System keychain → set to Always Trust
  • Windows: import into Trusted Root Certification Authorities
  • iOS / Android: open the file, install the profile, then enable full trust in settings
